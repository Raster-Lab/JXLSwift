# Examples

This directory contains example code demonstrating various features of JXLSwift.

## Running Examples

Examples can be run using Swift Package Manager:

```bash
swift run BasicEncoding
swift run QualityComparison
swift run HardwareDetection
swift run CustomConfiguration
```

## Available Examples

### BasicEncoding.swift
Demonstrates basic image encoding with default settings.

### QualityComparison.swift
Compares different quality settings and encoding modes.

### HardwareDetection.swift
Shows how to detect hardware capabilities and get optimization recommendations.

### CustomConfiguration.swift
Advanced encoder configuration with custom options.
